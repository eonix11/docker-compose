##Manual Installation
- Install nodejs locally
- Clone the repo
- Install dependencies
- Start the db locally
    - docker run -e POSTGRES_PASSWORD=mypass -d -p 5432:5432 postgres
    - go to neon.tech and get a new db
- Change .env file and update db creds
-  npx prisma migrate
- npx prisma generate
- npm run build 
- npm run start
